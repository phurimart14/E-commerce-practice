# Mini E-Commerce Project

## Coding Rules
- ใช้ TypeScript strict mode ห้ามใช้ any
- Component ใหม่ทุกตัวต้องมี explicit type สำหรับ props
- Server Components เป็น default, ใช้ "use client" เท่าที่จำเป็น

## Git Workflow
- ทุก feature ต้อง branch แยกจาก main: feature/<name>
- ก่อน commit ทุกครั้ง git add . ก่อน
- Commit message: feat:/fix:/chore:/docs:/refactor:
- ทำทีละ feature จบแล้วหยุดรอยืนยัน

## Communication
- คุยภาษาไทย เป็นกันเอง
- อธิบายสิ่งที่จะทำก่อนทำ
- ถ้าไม่แน่ใจให้ถาม อย่าเดา
